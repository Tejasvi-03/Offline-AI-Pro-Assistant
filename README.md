# 🤖 Offline AI Pro Assistant

Offline AI Pro Assistant is a Python-based application that works completely offline to analyze PDF and Excel files using AI. It allows users to upload documents, generate summaries, extract keywords, chat with their files, and visualize data through an interactive dashboard. The application uses local AI models to keep user data private.

## ✨ Features

- Extract text from PDF documents
- Generate AI-based document summaries
- Extract important keywords
- Chat with uploaded documents using Llama 3
- Analyze Excel and CSV files
- Interactive data visualization
- AI-generated insights
- Fully offline and privacy-focused

## 🛠 Technologies Used

- Python
- Streamlit
- Ollama (Llama 3)
- Transformers
- KeyBERT
- PyMuPDF
- Pandas
- Plotly

## 🚀 Installation

```bash
git clone https://github.com/Tejasvi-03/Offline-AI-Pro-Assistant.git
cd Offline-AI-Pro-Assistant
pip install -r requirements.txt
streamlit run app.py
```

> **Note:** Make sure Ollama is installed and the Llama 3 model is running before starting the application.
